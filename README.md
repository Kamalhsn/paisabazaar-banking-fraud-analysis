# Paisabazaar Banking Fraud Analysis

Exploratory Data Analysis project on Paisabazaar credit data

## Project Type
Exploratory Data Analysis (EDA)

## Objective
To analyze customer credit-related data and derive insights related to credit scores, risk behavior, and financial patterns.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
